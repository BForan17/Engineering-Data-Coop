# 📊 Engineering Data Sales Dashboard
**A Python-based data analytics tool for visualizing product performance.**

## 🚀 Overview
This project was developed to automate the analysis of engineering and sales data from Excel. It provides an interactive web interface to filter by product and region, allowing for quick insights into sales trends and regional performance.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Framework:** Streamlit
* **Libraries:** Pandas (Data Manipulation), Plotly (Interactive Charts)
* **Source:** Excel (`DataTest.xlsx`)

## ⚙️ How to Run Locally
1. **Clone the repo:**
   `git clone https://github.com/BForan17/Engineering-Data-Coop.git`
2. **Set up a Virtual Environment:**
   `python -m venv .venv`
3. **Activate Environment:**
   `.\.venv\Scripts\activate` (Windows)
4. **Install Dependencies:**
   `pip install -r requirements.txt`
5. **Launch App:**
   `streamlit run app.py`
